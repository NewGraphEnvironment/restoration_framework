# Restoration Framework

Knowledge base and planning hub for ecosystem restoration and conservation in British Columbia.

Memos are standalone R Markdown documents rendered to `docs/memos/` for GitHub Pages hosting. To add a new memo:

1. Create a directory under `memos/` (e.g., `memos/my-topic/`)
2. Write the `.Rmd` and render to HTML
3. Copy the rendered HTML to `docs/memos/my-topic/`
4. Add an entry to this table
